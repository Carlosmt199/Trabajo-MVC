# Trabajo-MVC
Instalar dependencias necesarias:
- bcrypt
- dotenv
- express
- pg

Crea una variable de entorno (.env) que incluya lo siguiente:

- DB_HOST=nombre_host
- DB_PORT=puerto_postgreSQL
- DB_USER=nombre_usuario
- DB_PASSWORD=tu_contrasena
- DB_NAME=nombre_base_de_datos
- PORT=puerto_servidor

(Nota: Nunca subas .env al repositorio; agrégalo a .gitignore)

Ejecutar con npm run dev
